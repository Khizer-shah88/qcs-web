# QCS Waterproofing Website

Professional waterproofing services website built with Next.js, React, and Tailwind CSS.

## Features

- 💧 Professional waterproofing services showcase
- 🏠 Modern, responsive design
- ⚡ Fast, optimized Next.js application
- 🎨 Beautiful UI with Framer Motion animations
- 📱 Mobile-first approach

## Tech Stack

- **Framework**: Next.js 13.5.1
- **UI Library**: React 18.2.0
- **Styling**: Tailwind CSS 3.3.3
- **Animations**: Framer Motion
- **Components**: Radix UI
- **Icons**: Lucide React

## Getting Started

### Prerequisites

- Node.js 18.x or higher
- npm (comes with Node.js)

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the website.

## Deployment

This project is configured for deployment on Vercel using npm.

### Deploy to Vercel

1. Push your code to GitHub
2. Import your repository to Vercel
3. Vercel will automatically detect Next.js and deploy

The deployment is configured to use:
- **Package Manager**: npm
- **Build Command**: `npm run build`
- **Install Command**: `npm install --legacy-peer-deps`
